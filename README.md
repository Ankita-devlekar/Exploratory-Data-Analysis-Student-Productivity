# Exploratory Data Analysis of Digital Distractions and Their Impact on Student Productivity

## Overview
This project analyzes a dataset of 20,000 student records to understand how digital distractions affect student productivity. The analysis includes data cleaning, visualization, and identifying patterns between screen time, stress levels, sleep patterns, study habits, focus scores, and productivity.

## Objective
- Analyze the impact of digital distractions on student productivity.
- Identify relationships between lifestyle factors and academic performance.
- Present insights through visualizations.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Key Analysis Performed
- Data Cleaning
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Data Visualization

## Dataset
The dataset contains approximately 20,000 student records with attributes such as:
- Screen Time
- Study Hours
- Stress Level
- Sleep Duration
- Focus Score
- Productivity Score

## Results
The analysis revealed meaningful relationships between screen time, stress, sleep quality, and student productivity, demonstrating how data analytics can support data-driven decision-making.
